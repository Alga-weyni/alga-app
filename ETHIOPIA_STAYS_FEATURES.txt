OVERVIEW
--------
Ethiopia Stays is a property rental platform specifically designed for the 
Ethiopian market. It connects property owners (hosts) with travelers seeking 
authentic Ethiopian accommodations, ranging from traditional homes to modern 
hotels and guesthouses.

USER ROLES & ACCESS CONTROL
================================================================================

FOUR USER ROLES:

1. GUEST/TENANT
   - Browse and book properties
   - Default role for new registrations

2. HOST/GUESTHOUSE OWNER
   - List and manage properties
   - Must be approved by admin

3. OPERATOR
   - Verify hosts and approve property listings
   - Review verification documents
   - Approve/reject properties

4. ADMIN
   - Full system control and user management
   - Can assign roles to users
   - Access to all system features

ROLE-BASED FEATURES:
- Automatic redirects based on role after login
- Role-specific navigation menus
- Role badge displayed in user profile dropdown
- Different dashboards for each role type

================================================================================
AUTHENTICATION SYSTEM
================================================================================

DUAL AUTHENTICATION OPTIONS:

1. PHONE + PASSWORD + OTP
   - Ethiopian phone format: +251XXXXXXXXX
   - 4-digit SMS OTP verification
   - OTP expires in 10 minutes
   - Auto-formats phone numbers
   - Development mode: OTP shown in console
   - Production ready: Can integrate Twilio or Ethiopian Telecom SMS

2. EMAIL + PASSWORD
   - Traditional email/password login
   - No OTP required for email users
   - Immediate account creation and login

SECURITY FEATURES:
- Bcrypt password hashing (10 salt rounds)
- Session-based authentication with PostgreSQL storage
- Secure session cookies (httpOnly, secure in production)
- Phone and email as unique identifiers
- Users can have either phone OR email (both supported)
- SESSION_SECRET environment variable required

UI COMPONENTS:
- Unified AuthDialog: Same dialog used everywhere on site
- Orange Phone & Email Tabs: Both tabs styled in orange
- OTP Verification Screen: For phone authentication
- Consistent authentication across all pages
- Role-based redirects after login
