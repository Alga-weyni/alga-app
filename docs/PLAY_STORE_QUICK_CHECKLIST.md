## üîë Build Production App (1-2 hours first time, 10 min after)

### First Time Setup
```bash
# 1. Generate upload keystore (SAVE PASSWORD SECURELY!)
cd android/app
keytool -genkey -v -keystore alga-upload-key.keystore \
  -alias alga-key-alias \
  -keyalg RSA \
  -keysize 2048 \
  -validity 10000

# 2. Backup keystore file to cloud storage (Dropbox, Google Drive, etc.)
# NEVER LOSE THIS FILE - you can't update your app without it!
```

### Every Release
```bash
# 1. Increment version in android/app/build.gradle
#    versionCode 1 ‚Üí 2 ‚Üí 3... (for each release)

# 2. Build signed AAB
cd android
./gradlew clean
./gradlew bundleRelease

# 3. Find your AAB here:
#    android/app/build/outputs/bundle/release/app-release.aab
```

**Full instructions:** `docs/BUILD_PRODUCTION_APK.md`

## üìû Quick Help

**Issue:** Can't find AAB file after build  
**Fix:** Check `android/app/build/outputs/bundle/release/`

**Issue:** Upload failed - signature mismatch  
**Fix:** Make sure you're using same keystore every time

**Issue:** Version code error  
**Fix:** Increment `versionCode` in `build.gradle`

**Issue:** Screenshots rejected  
**Fix:** Use exact size 1080 x 1920 px, PNG format

**Full troubleshooting:** `docs/GOOGLE_PLAY_SUBMISSION.md`
